7. การใช้งานตัวแปรใน Python

ใน Python ตัวแปรสามารถนำไปใช้ในหลายๆ วัตถุประสงค์ เช่น:

name = "Alice"
age = 25
is_student = False

# การใช้ตัวแปรในฟังก์ชัน
def greet():
    print("Hello, my name is", name)
    print("I am", age, "years old.")
    print("Am I a student?", is_student)

greet()

ในตัวอย่างนี้:

  • name ถูกกำหนดให้เก็บข้อความ
  • age ถูกกำหนดให้เก็บจำนวนเต็ม
  • is_student ถูกกำหนดให้เก็บค่า Boolean

เมื่อเรียกใช้ฟังก์ชัน `greet()`, จะได้ผลลัพธ์ดังนี้:

Hello, my name is Alice
I am 25 years old.
Am I a student? False
กลับไปที่หน้าแรก กลับไปที่บทเรียน
