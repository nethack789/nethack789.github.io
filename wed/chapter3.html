3. ประเภทของตัวแปร

ตัวแปรใน Python สามารถเก็บข้อมูลได้หลากหลายประเภท เช่น:

  • จำนวนเต็ม (Integer): ตัวเลขเต็ม
  • จำนวนทศนิยม (Float): ตัวเลขที่มีจุดทศนิยม
  • ข้อความ (String): ข้อความ
  • Boolean: ค่า True หรือ False
age = 25          # จำนวนเต็ม
price = 19.99     # จำนวนทศนิยม
name = "Alice"    # ข้อความ
is_active = True  # Boolean
กลับไปที่หน้าแรก กลับไปที่บทเรียน
